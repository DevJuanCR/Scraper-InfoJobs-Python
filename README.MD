# Scraper de Ofertas InfoJobs (Python)

Este proyecto es una herramienta de automatización desarrollada en **Python** que extrae ofertas de trabajo de "Desarrollador Junior" desde el portal InfoJobs de forma automática, sin necesidad de API oficial.

El objetivo principal es educativo: demostrar el uso de técnicas de **Web Scraping** dinámico, manejo de DOM y exportación de datos estructurados.
